# Arpit Yadav — Portfolio Website 🚀

Personal portfolio of **Arpit Yadav**, a Flutter Developer & Backend Developer.  
Built with React, TypeScript, Three.js, GSAP, and WebGL.
---

## About

This is the open-source version of my personal portfolio website.  
It showcases my skills as a **Flutter Developer** and **Backend Developer**, including:

- 🐦 Cross-platform mobile apps using Flutter & Dart
- ⚙️ Backend APIs with Node.js, FastAPI & Python
- 🔥 Firebase integration (Auth, Firestore, Cloud Functions)
- 🌐 Full-stack web apps with React & TypeScript

---

## Tech Stack

| Layer | Technologies |
|-------|-------------|
| Frontend (Portfolio site) | React, TypeScript, Vite |
| 3D & Animation | Three.js, GSAP, WebGL |
| Styling | CSS3 |
| My Dev Stack (showcased) | Flutter, Dart, Node.js, Express, FastAPI, MongoDB, PostgreSQL, Firebase |

---

## Running Locally

```bash
# Install dependencies
npm install

# Start dev server
npm run dev
```
